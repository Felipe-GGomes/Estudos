# 📅 Rotina Semanal — Felipe Gomes

> Dashboard pessoal de rotina semanal hospedado no GitHub Pages.  
> Sem backend, sem banco de dados — abre e usa.

**🔗 [felipe-ggomes.github.io/rotina](https://felipe-ggomes.github.io/rotina)**

---

## ✨ Funcionalidades

- **Abre no dia atual automaticamente**
- **Horário de início e fim** em cada bloco
- **Descrição expansível/retrátil** com o que fazer em cada atividade
- **Links diretos** para Coursera, HackerRank, GitHub, Lovable
- **Checkmarks** que salvam no `localStorage` — persistem até meia-noite (reset automático no próximo dia)
- **Calendário lateral** mostrando a trilha Coursera: Mês 1 → 4
- **Barra de progresso** semanal baseada nos checks
- **Botão de reset** manual do dia
- **100% responsivo** — funciona no celular

## 🚀 Como subir no GitHub Pages

```bash
# 1. Cria o repositório
git init
git add index.html README.md
git commit -m "feat: weekly routine dashboard"

# 2. Cria o repo no GitHub como "rotina" e faz push
git remote add origin https://github.com/Felipe-GGomes/rotina.git
git push -u origin main

# 3. Ativa o GitHub Pages
# Settings → Pages → Source: Deploy from branch → main → / (root) → Save

# Disponível em: https://felipe-ggomes.github.io/rotina
```
